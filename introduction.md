# Vibe Code Academy — Level 1

## Who this is for

You have never written a line of code. You have heard that AI can build apps for you and you want to find out if that is actually true — not for developers, but for people like you.

This level is for you if you:
- Have a project idea but assumed you needed to hire a developer
- Tried a tool like Cursor or Bolt once, got confused, and closed the tab
- Keep seeing people ship apps with AI and wonder what they know that you don't
- Are done watching tutorials and want to actually build something

This level is not for you if you are already writing code and looking for a productivity boost. This is for people starting from zero.

## What changes when you finish

By the end of Level 1 you will have built and deployed four real working apps:

1. **A personal landing page** — your first vibe-coded app, live at a URL
2. **A budget tracker** — income and expense tracking with categories, filters, and CSV export
3. **A movie explorer** — search real movies using a live API, save favourites
4. **A portfolio site** — showcasing all four projects, fully responsive, shareable

You will also be fluent in two AI coding tools: **Claude Code** (the terminal agent that manages your projects) and **Cursor** (the AI-powered editor you use for focused in-editor work). Every professional vibe coder uses both.

Most importantly, you will have a **documented personal workflow** — the exact steps you follow to go from idea to shipped app, written in your own words. That workflow is what you take into Level 2.

## The 15-day structure

Level 1 runs across 15 days, 3 hours per day.

| Week | Days | Focus |
|------|------|-------|
| Week 1 — Foundation | Days 1–5 | Claude Code, first app, core workflow, plugins |
| Week 2 — Tools & Skills | Days 6–10 | Cursor, debugging, deploy, responsive design, APIs |
| Week 3 — Build & Ship | Days 11–15 | Animation, Budget Tracker, Movie Explorer, Portfolio |

Work through one day per session. Do the steps — do not just read them. Each day ends with a concrete outcome. Do not move to the next day until you have that outcome.

## What you need

- A Claude account with Pro ($20/month) or Max ($100/month) — Claude Code does not work on the free plan
- A computer running Windows, macOS, or Linux
- 3 hours per session
- No prior coding experience

## The two tools you will use

**Claude Code** is the command-line version of Claude. You run it from your terminal. It reads your files, writes code, creates folders, and runs commands. Think of it as Claude with hands — it doesn't just give you code, it makes the changes.

**Cursor** is a code editor (based on VS Code) with AI built directly into it. Tab autocomplete predicts your next line as you type. Composer makes multi-file edits with a diff preview. Chat lets you ask questions about your code without leaving the editor.

You learn Claude Code first (Days 1–5) because it has less UI to navigate and teaches the core concepts clearly. You add Cursor at Day 6, and from there you use both together — each for what it does best.
